# Resume Location

Place your resume PDF file here and name it exactly:

```
resume.pdf
```

## Instructions

1. **Add your resume**:
   - Place your PDF resume in this folder (`/public/`)
   - Name it: `resume.pdf`
   - That's it! The site will automatically link to it

2. **File path will be**: `/public/resume.pdf`

3. **Users will access it at**: `https://yourdomain.com/resume.pdf`
